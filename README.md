# Interactive-Sales-Performance-Insights-Dashboard-Excel-

## 📌 Project Overview
This project presents an interactive Sales Dashboard built using Microsoft Excel to analyze business performance across regions, products, and time.

## 🚀 Key Features
- KPI Metrics (Total Revenue, Orders, Quantity, AOV)
- Interactive Filters using Slicers
- Sales Trend Analysis (Monthly)
- Region-wise and Product-wise performance

## 📈 Key Insights
- North region generates the highest revenue
- Electronics category dominates sales
- Cameras and Laptops are top-performing products
- High Average Order Value (~27K) indicates premium purchases

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations
- Slicers
